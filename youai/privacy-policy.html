Політика конфіденційності

Останнє оновлення: 2 лютого 2026 року

Ця Політика конфіденційності пояснює, як наш додаток для створення фото та відео за допомогою ШІ (надалі — "Додаток") обробляє ваші дані.

1. Збір та використання даних

Ми поважаємо вашу конфіденційність. Додаток розроблений таким чином, що він не збирає та не зберігає жодних персональних даних (таких як ім'я, адреса електронної пошти або номер телефону) на наших власних серверах.

  • Жодних персональних даних: Ми не вимагаємо реєстрації або створення облікового запису.
  • Згенерований контент: Ми не зберігаємо результати вашої генерації (фото та відео) на своїх серверах після завершення сесії передачі файлу на пристрій.
  • Вхідні дані: Фотографії, які ви додаєте для обробки ШІ, не зберігаються нами та використовуються лише для процесу генерації.

2. Сторонні сервіси (Обробка ШІ)

Для функціонування функцій штучного інтелекту ми використовуємо сторонній сервіс fal.ai.

  • Коли ви створюєте запит на генерацію, ваші вхідні дані (промпти або фото) передаються через захищене з'єднання до API fal.ai.
  • Ці дані використовуються виключно для виконання вашого запиту.
  • Ви можете ознайомитися з політикою конфіденційності нашого партнера тут: fal.ai Privacy Policy.

3. Дозволи пристрою

Для роботи Додатку потрібні наступні дозволи:

  • Фото та відео (Галерея): Для вибору фотографій, які будуть використані як база для генерації, та для збереження готових результатів на ваш пристрій.
  • Сповіщення (Push Notifications): Ми використовуємо сповіщення, щоб проінформувати вас про готовність відео/фото (оскільки генерація ШІ може тривати певний час), а також для повідомлень про нові пресети або функції. Ви можете вимкнути їх у будь-який момент у налаштуваннях пристрою.

4. Реклама та покупки (Треті сторони)

Ми використовуємо надійні сторонні сервіси для забезпечення роботи додатку:

  • Google AdMob: Використовується для показу реклами. AdMob може збирати технічні ідентифікатори пристрою (IDFA/AAID) для показу релевантних оголошень. Політика Google.
  • RevenueCat: Використовується для керування покупками та підписками. Ми використовуємо анонімний ідентифікатор RevenueCatUserID для відновлення ваших покупок без прив'язки до вашої особистої інформації. Політика RevenueCat.

5. Безпека контенту (AI Content)

Ми дотримуємося правил Google Play щодо контенту, створеного штучним інтелектом. Додаток забороняє створення незаконного або шкідливого контенту. Ми використовуємо вбудовані фільтри API для запобігання генерації неприйнятних матеріалів.

6. Видалення даних

Оскільки ми не зберігаємо ваші дані на наших серверах і не маємо облікових записів користувачів, видаляти немає чого. Щоб видалити всі локальні дані та кеш, просто видаліть Додаток зі свого пристрою. Для скидання рекламного ідентифікатора скористайтеся налаштуваннями Google у вашому телефоні.
